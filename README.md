# IpInfoga
A Tool used for Gathering IpAddress Information 

ABOUT:-
A tool which is used for Tracking and Getting the information regarding the Entered IpAddress.
It takes the Data from an API(Application programming interface)endpoint.
http:ip-api.com/json it is an Online IpTracker Which shows the Information in the form of json,xml etc.
This tool i.e IpInfoga Uses an json api endpoint which will directly communicate with the ip-api.com
